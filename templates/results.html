About Medical Symptom Search

Information Retrieval System for Medical Conditions

Purpose

This application is designed to help users identify possible medical conditions based on their symptoms. By entering symptoms into the search box, the system uses advanced information retrieval techniques to find matching conditions from its database.

Disclaimer: This tool is for informational purposes only and should not replace professional medical advice. Always consult with a healthcare provider for proper diagnosis and treatment.

Information Retrieval Methods

This application implements several IR techniques to provide accurate and helpful results:

BM25F Ranking

BM25F is a state-of-the-art ranking function that extends the classic BM25 algorithm to handle multiple fields. It incorporates:

  • Term Frequency (TF) - How often a query term appears in a document
  • Inverse Document Frequency (IDF) - Measures how important a term is
  • Document Length Normalization - Accounts for varying document lengths
  • Field Weighting - Allows different fields to have different importance
Text Processing Techniques

  • Fuzzy Matching - Handles misspellings and typos in search queries
  • Multi-field Searching - Searches across symptoms and synonyms fields
  • Spelling Suggestions - Offers alternative spellings for potentially misspelled terms

Technologies Used

Python Flask Whoosh Bootstrap HTML/CSS JavaScript

Whoosh is a pure Python search engine library that provides indexing and searching capabilities. It's used in this application for creating a searchable index of medical conditions and symptoms, allowing for fast and relevant search results.

Database

The application includes information on 25+ common medical conditions, including:

  • Common cold and influenza
  • Respiratory conditions (asthma, bronchitis, pneumonia)
  • Digestive disorders (GERD, IBS)
  • Chronic conditions (hypertension, diabetes)
  • Mental health (depression, anxiety)
  • And many more

Each condition is indexed with its symptoms, description, synonyms, severity, and treatment information.
